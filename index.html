Accueil - Bienvenue

 

Je mets à disposition sur ce site les exercices que je crée avec le logiciel : Jclic ,

exercices d'aide à la lecture pour des enfants ou jeunes présentant des difficultés d'apprentissage de la lecture.

Ils peuvent être un outil de soutien et de remédiation en lecture et écriture pour des jeunes dyslexiques.

Ils peuvent permettre à des jeunes autistes d'entrer dans les apprentissages.

Ils peuvent venir en complément de l'apprentissage de la lecture pour des élèves de CP ou de CE1.

Vous pouvez les utiliser en ligne en cliquant sur "lancer l'application" ou les "télécharger" sur votre ordinateur pour les utiliser sans être connecté à Internet.

 

Pour utiliser les exercices en ligne, téléchargez le logiciel Java de Sun Microsystems

To use exercises online, you need install Java

Il va s'installer automatiquement sur votre ordinateur si vous autorisez l'installation.

 

Pour utiliser les exercices sur votre ordinateur, téléchargez Jclic Player

To use exercises if you download them on your computer, you need install Jclic Player

     

    Installez Jclic Player. Voici une aide :

 

Les activités sont classées en trois niveaux :

"Lire niveau 1" correspond à des lecteurs déjà avancés dans l'apprentissage

"Lire niveau 2" correspond à des lecteurs débutants dans l'apprentissage

"Pré-lecture" correspond à des enfants ou jeunes non lecteurs

 

Certaines activités utilisent la police "BV Rondes" créée par Bernard VIVIER à téléchargeret à installer sur votre ordinateur. Une nouvelle police est utilisée. Télécharger "Cursive standard"
